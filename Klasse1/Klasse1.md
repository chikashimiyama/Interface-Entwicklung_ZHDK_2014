## Multitouch

![fingerpinger](img/fingerpinger.png)

- Website von [Fingerpinger](http://www.anyma.ch/2009/research/multitouch-external-for-maxmsp/)

Mit Fingerpinger kann man folgende Fingerdaten auf dem Trackpad bekommen. 

- Frame
- Angle
- Semi-Major-Axis
- Semi-Minor-Axis
- Xpos
- Ypos
- VelocityX
- VelocityY
- Identifier
- State
- Size
